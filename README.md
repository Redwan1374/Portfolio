## 🚀 Getting Started

To run this project locally, make sure **Node.js** and **npm** are installed on your computer.

### 1. Clone the repository

```bash
git clone https://github.com/Redwan1374/Portfolio.git
```

### 2. Go to the project directory

```bash
cd Portfolio
```

### 3. Install dependencies

```bash
npm install
```

### 4. Start the development server

```bash
npm run dev
```

Then open the local URL displayed in the terminal.

---

## 📦 Production Build

Create an optimized production build using:

```bash
npm run build
```

Preview the production build locally:

```bash
npm run preview
```
